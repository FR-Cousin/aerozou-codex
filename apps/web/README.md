# @aerozou/web

Prototype **PWA sans build** (HTML/CSS/JS) pour itération rapide.

## Lancement local depuis la racine du repo

```bash
npm run web:serve
```

Puis ouvrir `http://localhost:5173`.

## Lancement local depuis le dossier `apps/web`

```bash
python -m http.server 5173
```

> L'application appelle l'API sur `http://localhost:8080/api`.
