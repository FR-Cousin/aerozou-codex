# Roadmap de développement

## Cycle 1 — Prototype VFR (actuel)

- Monorepo + API TypeScript initialisée.
- Calcul de route VFR (distance, cap, GS, temps, carburant).
- Décodage METAR basique.
- Documentation initiale.

## Cycle 2 — UX + données temps réel

- UI web PWA: planificateur cartographique, fiche route, météo/NOTAM.
- Connecteurs OpenAIP/OurAirports/NOAA.
- Authentification, profils pilote, logbook v1.

## Cycle 3 — Mobile + coffre-fort + monétisation

- App mobile React Native.
- Coffre-fort documents sécurisé.
- Essai gratuit + dons/achat unique + ads discrètes (feature flag).

## Cycle 4 — IFR et communauté

- SID/STAR/approches, enrichissement calculs IFR.
- Fonctionnalités communautaires (notes, fil d'activité).
- Assistant IA enrichi (explication de route/check-lists vocales).
